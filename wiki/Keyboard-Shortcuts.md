## Operation Shortcuts

### Primary Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+S` | **Scan** | Start disk scan |
| `Ctrl+Shift+A` | **Analyze** | Start deep analysis |
| `Ctrl+Shift+F` | **Format** | Format disk |
| `Ctrl+R` | **Restore** | Start recovery |
| `Ctrl+B` | **Batch Verify** | Start batch verification |

### Operation Control

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Escape` | **Cancel** | Cancel current operation |
| `Space` | **Pause/Resume** | Pause or resume operation |

---

## Device Control

### Connection

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Shift+C` | **Connect/Disconnect** | Toggle device connection |

### Motor and Seek

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+M` | **Motor Toggle** | Turn motor on/off |
| `Ctrl+0` | **Seek Track 0** | Return head to track 0 |
| `Ctrl+J` | **Eject** | Eject disk (if supported) |

---

## Navigation

### Tab Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+1` | Overview tab |
| `Ctrl+2` | Flux tab |
| `Ctrl+3` | Errors tab |
| `Ctrl+4` | Diagnostics tab |
| `Ctrl+5` | Recovery tab |

### Sector Map

| Shortcut | Action |
|----------|--------|
| `Arrow Keys` | Move selection |
| `Home` | First sector |
| `End` | Last sector |
| `Page Up` | Previous track |
| `Page Down` | Next track |

---

## View Controls

### Zoom

| Shortcut | Action |
|----------|--------|
| `Ctrl++` or `Ctrl+=` | Zoom in |
| `Ctrl+-` | Zoom out |
| `Ctrl+0` | Fit to window |

### Display

| Shortcut | Action |
|----------|--------|
| `F11` | Toggle fullscreen |
| `Ctrl+L` | Toggle legend |

---

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+O` | **Open Image** | Load disk image file |
| `Ctrl+E` | **Export** | Export disk to image |
| `Ctrl+Shift+R` | **Export Report** | Generate report |
| `Ctrl+Shift+S` | **Save As** | Save with new name |

---

## General

### Application

| Shortcut | Action |
|----------|--------|
| `Ctrl+,` | Open Settings |
| `F1` | Help/Documentation |
| `Ctrl+Q` or `Alt+F4` | Quit application |

### Edit

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Copy (sector data) |
| `Ctrl+A` | Select all |
